# HTML Trackers

This folder contains a small set of standalone HTML pages plus a landing page that links them together.

Files:
- `index.html` — landing page with quick-link cards to every tracker
- `ai-product-engineer.html` — six-week startup job-search and shipping roadmap
- `auth.html` — auth study checklist covering JWT, refresh tokens, and security pitfalls
- `backend.html` — backend mastery roadmap for FastAPI, PostgreSQL, and project delivery
- `workout.html` — recomposition workout tracker with timers and weekly history

Notes:
- The pages are self-contained and can be opened directly in a browser.
- `auth.html`, `backend.html`, and `workout.html` now use a 25% larger typography scale.
- SEO metadata and section-level summaries were added to improve discoverability and navigation.
