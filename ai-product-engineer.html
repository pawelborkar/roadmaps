The Project
DevPulse — AI-powered Developer Activity Summarizer

What you'll build

  • FastAPI backend with GitHub OAuth
  • Fetches your GitHub commits, PRs, activity
  • Sends data to Claude/OpenAI API for summarization
  • Returns a "what did I build this week" digest
  • Simple React or SvelteKit frontend
  • Deployed live (Railway or Render, free tier)

Why this project specifically

  • Closes FastAPI gap — real backend from scratch
  • Closes AI integration gap — LLM API in production
  • Genuinely useful — you'll actually use it
  • Impressive to devtools startups specifically
  • Small enough to finish in 2 weeks
  • Deployable = verifiable by interviewers
